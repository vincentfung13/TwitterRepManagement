About the Project

About me

My name is Vincent, I am a final year Software Engineering student in the University of Glasgow, UK. Before coming to the UK I spent two years majoring Electronics Engineering in Sun Yat-Sen University, mainland of China.

This system is my undergraduate final year project in Glasgow University supervised by Dr. Iadh Ounis. It aims to help companies to keep track of the online reputation of their products, as well as individual users to monitor products they are interested in.

Contributing

The back-end of the system is written in Python and it is built using the Django Framework. Note that it has a dependency of PostgresSQL (version 9.3 or later) and Django version 1.9, for using of the JSONField with Django ORM.

I have been developing this system for around four months, therefore it is far from perfect and can be scaled. The project is open-sourced on Github. You can open a pull request for contributing to the project.
